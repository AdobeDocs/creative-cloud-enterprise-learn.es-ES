# &#x200B;3. Crea tu primer gráfico

Una vez que sepa qué es un nodo, conexión y plantilla, estará listo para crear su primer flujo de trabajo.

1. **Abra un lienzo en blanco o empiece desde una plantilla.** Examine la biblioteca de nodos o la galería de plantillas para encontrar un punto de partida cercano a lo que necesita.
2. **Conectar nodos para generar el flujo de trabajo.** Dibuje conexiones desde la salida de un nodo a la entrada del siguiente nodo para encadenar pasos juntos; por ejemplo, una imagen de entrada que fluya a un nodo de estilo, luego a un generador de variaciones y, a continuación, a la exportación.
3. **Agrega controles de entrada para cualquier cosa que desees cambiar más adelante.** Convierte valores estáticos en reguladores, selectores de color o campos de texto para que tú (o un compañero de equipo) podáis ajustar los parámetros clave sin editar el gráfico subyacente.
4. **Ejecute el gráfico y obtenga una vista previa de cada paso.** Puede obtener una vista previa del resultado en cualquier nodo antes de que finalice la ejecución del flujo de trabajo completo.
5. **Cambiar una entrada y volver a ejecutar.** Cambia una imagen de referencia, ajusta una petición de datos de estilo o ajusta una variable: todo lo que ocurre después se actualiza automáticamente. Nada se reconstruye a mano.

## Qué probar primero

Si no estás seguro de por dónde empezar, abre una de las plantillas listas para usar en la biblioteca de plantillas en lugar de un lienzo en blanco; es la forma más rápida de ver un gráfico de trabajo completo antes de crear uno desde cero.
